# Web Workspace

This folder contains the authenticated web client assets.

## Entrypoint
- HTML shell: `app.html`

## Source layout
- Styles: `web/css`
- JavaScript: `web/js`

## Local run
```bash
cd <repo-root>
python3 -m http.server 5173
```

Open `http://localhost:5173/app.html`.

For credential setup and validation scripts, see:
- `docs/current/web/web_and_node_testing.md`
